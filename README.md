# Wanderlust â€“ Full Stack Travel Application

Wanderlust is a full-stack travel web application that allows users to explore, create, and manage travel listings with reviews and ratings.

## Features
- User authentication and authorization (login/signup)
- Create, edit, and delete travel listings
- Add reviews and ratings for listings
- Image upload and management using Cloudinary
- Flash messages for user feedback
- Responsive UI for better user experience

## Tech Stack
**Frontend**
- HTML
- EJS
- Tailwind CSS
- javaScript

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose

**Other Tools & Libraries**
- Passport.js (Authentication & Authorization)
- Cloudinary (Image storage)
- Multer (File uploads)

## Project Highlights
- Implemented RESTful APIs for listings and reviews
- Secure routes using middleware (authentication & authorization)
- Followed MVC architecture for better code structure
- Optimized database queries using Mongoose

## Use Case
Wanderlust helps users discover travel destinations, share experiences, and manage travel-related content in a secure and scalable way.
